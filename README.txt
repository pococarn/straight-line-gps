Farm A-B Lightbar V7 Field Mode with Logo

Upload these files to GitHub root:
- index.html
- lanteri-logo.png
- apple-touch-icon.png

New features:
- 13 LED lightbar
- Lanteri Enterprises logo in bottom-right corner
- Smaller GPS accuracy meter in top-right corner
- Simplified landscape no-scroll field layout
- Start GPS, Set A, Set B, Nudge, Width, Presets, Menu
- Screen keep-on helper button
- Width presets in menu
- Full-screen Home Screen app mode

Use on iPad:
1. Replace the files on GitHub.
2. Wait 1-2 minutes.
3. Open https://pococarn.github.io/farm-ab-lightbar/ on the iPad.
4. Share -> Add to Home Screen.
5. Open from the Home Screen icon.
6. Turn iPad landscape.

For old iPad screen staying awake:
Settings -> Display & Brightness -> Auto-Lock -> Never
