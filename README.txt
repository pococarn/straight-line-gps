iPad A-B Lightbar

This is a simple offline HTML/JavaScript A-B line navigator for an old iPad mini cellular.

How to use:
1. Open index.html on the iPad in Safari.
2. Press START GPS.
3. Press SET A.
4. Drive forward 20 to 50 metres.
5. Press SET B.
6. Follow the big arrows.

Important:
- iPad mini 1 cellular has built-in GPS.
- GPS accuracy may only be a few metres.
- This is for rough farm guidance, not precision RTK.
- Works best with the iPad mounted where it has a clear sky view.
- Safari may require location permission.
- If opening the file directly does not allow GPS, host it from a local web server or put it on a simple HTTPS page.
